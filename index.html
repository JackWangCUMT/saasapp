SaasApp by yatendra

SaasApp is a starter kit for creating a software as a service (SAAS) application using ASP.Net, JQuery and MySQL. It uses Twitter's bootstrap framework for layout.

SaasApp is a starter kit for creating a multi-tenant software as a service (SAAS) application using ASP.Net, JQuery and MySQL using shared database architecture. It uses Twitter's bootstrap framework for layout.

What does it do

  • Creates a basic site for the app with a home page and page that displays various plans.
  • User can then select a plan and register an account with selected plan specifying administrator password among other details.
  • Each account say abccorp gets a sub-domain abccorp.[appdomain]
  • [appdomain] can be a top level domain like myapp.com or localhost url (like localhost, localhost:3203 etc) or a subdomain like app.mydomain.com
  • administrator can then login using specified password and create more users that can also access the same app at abccorp.[appdomain] and their access is limited to abccorp account only
  • Administrator can manage users (list, add, edit and delete).
